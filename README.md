### Development

Take git repository:
```sh
$ git clone ssh://git@git.milosolutions.com:8922/web/Bctpython.git
```

Run vagrant:
```sh
$ vagrant up
```

Connect to vagrant:
```sh
$ vagrant ssh
```

Run dev server:
```sh
$ rs
```
